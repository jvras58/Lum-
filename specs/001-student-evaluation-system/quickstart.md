## Prerequisites

| Tool | Version | Notes |
|------|---------|-------|
| Node.js | ≥ 20.x LTS | Required for Next.js 16 |
| pnpm | ≥ 9.x | Package manager |
| Docker | any | PostgreSQL via docker-compose |
| PostgreSQL client | any | Optional; for direct DB inspection |

---

## Initial Setup

```bash
# 1. Install dependencies
pnpm install

# 2. Start PostgreSQL
docker compose up -d db

# 3. Copy environment variables
cp .env.example .env.local
# Edit .env.local:
#   DATABASE_URL=postgresql://postgres:postgres@localhost:5432/lum_dev
#   SECRET_CRON_TOKEN=<generate with: openssl rand -hex 32>
#   EMAIL_PROVIDER=smtp  # or resend
#   SECRET_SMTP_HOST=...
#   SECRET_SMTP_USER=...
#   SECRET_SMTP_PASS=...

# 4. Generate and apply migrations
pnpm drizzle-kit generate
pnpm drizzle-kit migrate

# 5. (Optional) Seed development data
pnpm db:seed

# 6. Start development server
pnpm dev
# → http://localhost:3000
```

## Key Development Workflows

### Adding a new Server Action

```typescript
// src/actions/example.ts
"use server"
import "server-only"
import { db } from "@/db"
import { revalidatePath } from "next/cache"
import type { ActionResult } from "@/lib/types"

export async function createExample(input: unknown): Promise<ActionResult<Example>> {
  const parsed = exampleSchema.safeParse(input)
  if (!parsed.success) return { success: false, error: parsed.error.message }

  // assertAuthenticated() — placeholder for future auth
  const start = Date.now()
  try {
    const [row] = await db.insert(examples).values(parsed.data).returning()
    revalidatePath("/examples")
    console.info(JSON.stringify({ action: "createExample", duration: Date.now() - start, success: true }))
    return { success: true, data: row }
  } catch (err) {
    console.error(JSON.stringify({ action: "createExample", error: String(err) }))
    return { success: false, error: "Erro ao criar registro" }
  }
}
```

### Running migrations

```bash
# After editing src/db/schema.ts:
pnpm drizzle-kit generate   # generates SQL migration
pnpm drizzle-kit migrate    # applies to dev DB

# Check SQL before applying:
pnpm drizzle-kit generate --dry-run
```

### Running integration tests

```bash
# Requires DB running (docker compose up -d db)
DATABASE_URL=postgresql://postgres:postgres@localhost:5432/lum_test pnpm test:integration
```

### Triggering email consolidation manually (dev)

```bash
curl -X POST http://localhost:3000/api/cron/consolidate-emails \
  -H "Authorization: Bearer <SECRET_CRON_TOKEN from .env.local>" \
  -H "Content-Type: application/json" \
  -d '{"date": "2026-04-19"}'
```

### Adding a new Shadcn component

```bash
npx shadcn@latest add <component-name>
# Component appears in src/components/ui/
```

## Environment Variables Reference

| Variable | Required | Description |
|----------|----------|-------------|
| `DATABASE_URL` | Yes | PostgreSQL connection string |
| `SECRET_CRON_TOKEN` | Yes | Bearer token for cron endpoint |
| `EMAIL_PROVIDER` | Yes | `smtp` or `resend` |
| `SECRET_SMTP_HOST` | If smtp | SMTP server hostname |
| `SECRET_SMTP_PORT` | If smtp | SMTP port (default 587) |
| `SECRET_SMTP_USER` | If smtp | SMTP auth username |
| `SECRET_SMTP_PASS` | If smtp | SMTP auth password |
| `SECRET_RESEND_API_KEY` | If resend | Resend API key |
| `EMAIL_FROM` | Yes | Sender address (non-secret) |
